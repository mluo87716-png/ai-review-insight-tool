# Metrics Framework

## 北极星指标

Weekly Adopted Conversion Insights

每周被运营人员采纳、并进入卖点表达、内容排期、详情页优化或客服 FAQ 的 AI 洞察数量。

这个指标比“生成次数”更重要，因为 AI 产品运营关注的是 AI 输出是否真正帮助运营吸引用户选择产品。

## 核心漏斗

| 阶段 | 指标 | 解释 |
| --- | --- | --- |
| 输入 | 评论导入率 | 用户是否愿意把评论交给工具处理 |
| 生成 | 评论分析完成率 | 工具是否能稳定输出报告 |
| 判断 | 人工复核完成率 | 用户是否愿意检查 AI 输出 |
| 采纳 | 洞察采纳率 | 输出是否被写进周报、Brief 或复盘 |
| 转化 | 内容选题转化率 | 选题是否进入发布排期 |
| 转化 | 详情页优化采纳率 | 洞察是否进入商品详情页或活动页 |
| 服务 | 客服 FAQ 覆盖率 | 高频疑虑是否被提前回答 |
| 留存 | 7 日复用率 | 工具是否有持续使用价值 |

## 质量指标

- 人工修正率
- 错误分类率
- 无效洞察占比
- 重复输出占比
- 用户满意度评分

## 业务价值指标

- 评论整理耗时下降
- 周报制作耗时下降
- 内容选题产出数量提升
- 详情页优化建议数量
- 负反馈发现速度提升
- 用户购买阻碍识别数量
- 可放大卖点沉淀数量

## MVP 目标

| 指标 | 目标 |
| --- | --- |
| 评论分析完成率 | 80% |
| 洞察采纳率 | 30% |
| 内容选题转化率 | 20% |
| 详情页优化采纳率 | 20% |
| 客服 FAQ 覆盖率 | 30% |
| 人工修正率 | 低于 40% |
| 7 日复用率 | 25% |

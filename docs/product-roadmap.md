# Product Roadmap

## 当前版本：V0.1 Demo

目标：让 HR、面试官或业务同学 30 秒内理解项目价值。

已完成：

- CSV 样例评论分析
- 演示模式，无需 API Key
- Claude API 扩展模式
- Markdown 洞察报告
- 本地产品 Demo
- PRD、用户画像、指标体系、Prompt 模板

## V0.2：运营工作台

目标：让工具更像真实运营工作台。

计划功能：

- CSV 上传入口
- 情绪和主题可视化图表
- 代表性用户原话筛选
- 一键复制周报总结
- 一键复制小红书/抖音选题

## V0.3：人工复核闭环

目标：让 AI 输出质量可评估。

计划功能：

- 给每条洞察标记“采纳/修改/不用”
- 记录人工修正原因
- 统计人工修正率
- 沉淀高质量 Prompt 模板

## V0.4：竞品对比分析

目标：从单品评论分析扩展到竞品分析。

计划功能：

- 支持多个商品评论文件
- 对比不同商品的高频痛点
- 对比竞品卖点和负反馈
- 生成差异化内容策略建议

## 30 天迭代计划

| 时间 | 目标 | 交付 |
| --- | --- | --- |
| 第 1 周 | 完成基础工具和报告 | Python 脚本、样例数据、报告 |
| 第 2 周 | 完成作品集展示 | README、Demo、PRD、指标体系 |
| 第 3 周 | 增强产品化能力 | 可视化看板、复制周报、选题卡片 |
| 第 4 周 | 增强运营闭环 | 人工复核、采纳率、复盘模板 |

## 长期方向

- 接入平台公开评论数据
- 支持更多行业标签体系
- 接入 Dify / Coze 构建低代码 AI 工作流
- 支持团队协作和历史洞察沉淀
